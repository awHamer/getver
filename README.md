## npm-pkg-version

Lightweight cli tool for getting currently installed package version. 

## Installation

```bash
$ npm install -g npm-pkg-version
```
or Yarn
```bash
$ yarn global add npm-pkg-version
```

## Example of usage

```bash
$ npmver express
$ npmver global npm
```

Output

```
 -> Package Name:  npm  (global)
 -> Package version:  6.4.1
```
