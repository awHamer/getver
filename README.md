## npm-pkg-version

A lightweight cli tool for getting proper version of npm package both locally and globally 

## Installation

```bash
$ npm install -g npm-pkg-version
```
or Yarn
```bash
$ yarn global add npm-pkg-version
```

## Example of usage

```bash
$ npmver express
$ npmver global npm
```

Output

```
 -> Package Name:  npm  (global)
 -> Package version:  6.4.1
```
